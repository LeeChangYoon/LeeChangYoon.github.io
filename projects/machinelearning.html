I have discussed the basic concepts that are used to implement the linear regression model, specifically gradient descent and least square. Then, I have explained how to evaluate the performance of the prediction model by assessing the mean squared error (MSE). After discussing all the concepts, I have presented the implemented Python code for task 1 and task 2, representing each model that applies gradient descent and the least square method of the linear regression model. In the results, I have shown the prediction graph based on the given data set and the coefficients for the predicted model. Additionally, I have made comparisons between the built-in linear regression models and the implemented models.

After discussing all the concepts, I have presented the implemented Python code for task 1 and task 2, which involves training the model with the training dataset and classifying the data in the test dataset using the trained model. In the results, I have shown the prediction graph based on the given dataset and the coefficients for the predicted model. Furthermore, I have made comparisons between the built-in logistic regression models and the implemented models.
